# Make Talking Head Storyboard

一个为中文口播视频制作 B-roll 分镜的 Codex Skill。

它会理解整篇文案的上下文，判断哪些句子适合加入 B-roll，并输出可以直接执行的分镜表和中文 AI 视频提示词。画面既可以复现文中的场景，也可以让固定的 AI 动画主人公参与正面或负面情景演绎。

默认适用于知识口播、老板 IP 和品牌宣传；生成建议以 `1:1`、`5-10 秒`的即梦或可灵 AI 视频为目标。

```text
使用 $make-talking-head-storyboard，为下面的口播稿制作 B-roll 分镜执行方案：

[粘贴完整口播稿]
```

## English

A Codex Skill for creating B-roll storyboards for Chinese talking-head videos.

It understands the full script, decides which lines benefit from B-roll, and produces an executable storyboard with context-aware Chinese AI-video prompts. Scenes can recreate events from the script or feature a recurring AI animated protagonist performing positive or negative roles.

It is designed for knowledge explainers, founder or executive IP videos, and brand promotion, with prompts targeting `1:1`, `5-10 second` Jimeng or Kling videos.
